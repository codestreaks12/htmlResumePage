Project Name: HTML Resume Page

A simple static resume page website build using only html and deployed using vercel

Live Demo: https://html-only-resume.vercel.app/

Tech Stack: HTML

Features:
-Single Page Website
-Good foundation to showcase HTML Semantic website building
-Clear and Clean UI matches real world word resume

**Setup**
**Option 1: **
1. Download or clone the repository
2. Open index.html in your browser

**Option 2: **
1. Download or clone repository
2. Open in VS Code
3. Install extension live server
4. Open the index.html page and click on go live on the bottom of the screen
